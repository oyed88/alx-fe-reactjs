# React Query Demo

This project demonstrates advanced data fetching and state management in React using React Query.

## Features

- Fetch posts from JSONPlaceholder API
- Display loading and error states
- Cache fetched data automatically
- Manually refetch data using a button

## Technologies Used

- React
- Vite
- React Query

## API Used

https://jsonplaceholder.typicode.com/posts

## Getting Started

1. Install dependencies:

npm install

2. Run the development server:

npm run dev
